Laziness & Granularity

Laziness

Given I have a user named "admin"

And the user named "admin" authenticates

And the user named "admin" is authorized to add clients

When the user visits the add new client page

And the user provides new client data

And the user clicks on Submit button

Then the user should see a new client added message

Granularity

Given I am a user

When I buy a product

Then I should see the checkout page
